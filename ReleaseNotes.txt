v1.6.1
- Apply the release version consistently to NuGet, assembly, file, and informational metadata.
- Validate every shipped DLL's embedded version before publishing a package.

v1.6.0
- Add opt-in unsafe generated accessors for callers that guarantee valid padded buffers.
- Add allocation-free contiguous and strided batch primitives plus generated batch accessors.
- Isolate source generation per bit object and resolve backing storage through Roslyn symbols.
- Support nullable byte arrays, escaped identifiers, compatible derived attributes, and fully qualified generated references.
- Add BITSKIT007-BITSKIT014 diagnostics for invalid generator options, layouts, names, backing fields, pointers, and modifiers.
- Fix nonzero-offset array-backed accessors and unsafe raw-pointer generation.

v1.5.0
- Add sequential BitStreamWriter support for non-seekable compression, encryption, and network streams.
- Generate direct mask-and-shift scalar accessors for integral, enum, and Boolean fields.
- Specialize aligned Memory, Span, array, and byte InlineArray accessors plus common 11-, 12-, 24-, and 48-bit fields.
- Accelerate buffered stream operations while preserving seeking, EOF, and in-place behavior.
- Add normalized generated-accessor benchmarks and an original-version performance regression workflow.
- Fix signed integral-backed Boolean getters.

v1.4.0
- Make stream reads reliable across partial reads and fail cleanly at EOF.
- Add pooled sequential buffering for BitStreamReader and BitStreamWriter.
- Preserve in-place writes and support stream positions beyond Int32.MaxValue.
- Improve aligned UInt32 stream writes by about 15% and mixed unaligned writes by about 6% in local benchmarks.

v1.3.1
- Fix tag-driven GitHub Release creation in the split release pipeline.

v1.3.0
- Establish community-maintained releases under the RejectKid.BitsKit package ID.
- Add .NET 10 support while retaining existing library targets.
- Add reproducible builds, dependency locking, and automated package validation.
- Fix incremental source-generator caching and restore the BITSKIT003 diagnostic.
- Fix inline-array setters for modern C# compiler span overload resolution.
- Remove EOL .NET 6 and .NET 7 targets and modernize test and benchmark dependencies.
- Add categorized local and automated benchmark reporting for the library's features.
